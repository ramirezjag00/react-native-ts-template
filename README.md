# React Native TS Template

> Read template/README.md after using

## To use:

```
npx react-native init AwesomeApp --template https://github.com/ramirezjag00/react-native-ts-template.git 
```

## Template Set up
- react-native >= v0.66
- typescript >= 4.1.0
- eslint
- prettier
- path alias
- husky
- lint-staged
- Hermes enabled for iOS and Android
